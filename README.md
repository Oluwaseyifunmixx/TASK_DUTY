# Task Duty - Frontend

Built with React + TypeScript + Vite

## Prerequisites
- Node.js

## Run the Frontend
```bash
npm install
npm run dev
```

App runs on http://localhost:5173

## Features
- Landing page
- My Tasks page with filtering
- New Task form with validation
- Edit Task form with validation
- Responsive design

## Tech Stack
- React, TypeScript, React Router, Axios
